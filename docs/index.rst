Overview
--------

A fully compliant external provider for `GARM <https://github.com/cloudbase/garm>`_
that provisions GitHub/Gitea runners on Proxmox VE.

Supports creating runners as either **QEMU Virtual Machines** (via cloud-init and
QEMU Guest Agent) or **LXC Containers** (via exec injection).
